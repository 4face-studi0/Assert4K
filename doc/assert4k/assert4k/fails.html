fails

Asserts that given function block fails by throwing an exception. ` assert that fails { throw Exception() } `

Return

An exception that was expected to be thrown and was successfully caught. The returned exception can be inspected further, for example by asserting its property values.Note: suspend functions cannot be called in this block because block is not inlined, use coFails instead.
fun fails(block: () -> Unit): AssertionContinuation<Throwable>
